

Steps to Execute:


1. Open Eclipse.

2. Create a new Java Project in Eclipse. 

3. Import the "Student Testing" files to the project.

4. Right click on the existing project -> go to properties -> Java Build Path -> Libraries.

5. Click on "Add external Jars" and select the "TaskExecutor.jar" file.

6. Now right click on "TaskExecutorTest.java" file which contains the main class and run the java project.


Note: The source files are present in /src directory.
